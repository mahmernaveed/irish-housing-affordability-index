# irish-housing-affordability-index
Open Data project — composite housing affordability index across Irish regions
# This project investigates whether a composite housing affordability index can be 
constructed using open data across Irish regions. Using publicly available datasets 
from the CSO (income), RTB (rent), and the Property Price Register (property prices), 
the project follows a reproducible Open Data workflow aligned with FAIR principles. 
The index will rank Irish counties by affordability and identify regional disparities, 
contributing to evidence-based understanding of Ireland's housing crisis.
